# Colibrí Creativo

Proyecto de tienda en línea desarrollado con React.

## Descripción
Colibrí Creativo es una plataforma web para la venta de productos personalizados para eventos y regalos especiales.

## Funcionalidades
- Catálogo de productos
- Carrito de compras
- Personalización de productos
- Página de contacto

## Tecnologías utilizadas
- React
- JavaScript
- HTML
- CSS
